# Lab 1 - UI Design

## Project Description
This lab is an introductory exercise in web development using HTML and CSS. It focuses on understanding the fundamentals of creating a basic web page, including structure, layout, and simple styling.

The objective of this lab is to build a strong foundation in web development concepts before progressing to more complex projects.

---

## SDG Theme
This project aligns with:

**SDG 4: Quality Education**

It supports learning by:
- Introducing basic web development skills
- Building understanding of HTML structure and CSS styling
- Encouraging hands-on practice in creating web pages

---

## Features
- Basic HTML page structure
- Simple CSS styling
- Introduction to web layout design
- Clean and minimal interface
- Beginner-friendly implementation
